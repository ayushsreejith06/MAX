# MAX Project

Full-stack application with frontend, backend, smart contracts, and shared utilities.

## Project Structure

```
/max
  /frontend    - Next.js 15 app
  /backend     - Node/Express API
  /contracts   - Solidity contracts
  /shared      - Types, schemas, utils
  /docs        - Documentation
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
